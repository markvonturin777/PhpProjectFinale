About

Progetto di programmazione in PHP realizzato in classe con il professor Enrico Zimuel.


Realizzazione di un'applicazione PHP per la gestione di un quotidiano online con i seguenti requisiti tecnici:

  • Utilizzo del pattern SimpleMVC: github.com/ezimuel
  • Utilizzo del template Twig
  • Memorizzazione degli articoli in un database (MySQL o SQLite) utilizzando PDO di PHP per l'accesso
  • Sistema di gestione articoli con operazioni CRUD
  • Utilizzo di test unitario con PHPUNIT
